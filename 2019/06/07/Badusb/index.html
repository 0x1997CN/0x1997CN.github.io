Badusb

2019-06-07 /
  • hardware

  BadUsb是一种可以伪装成键盘输入的恶意设备，Badusb是usb的一款严重漏洞，攻击者可利用该缺陷，在有效且不被检测到的情况下，悄悄地植入恶意软件

0x01 环境基础

  • Digispark 开发板（某宝￥9.9）
  • Digispark驱动
  • Arduino IDE
  • 科学上网（你懂的）

0x02 环境搭建

1、安装 Arduino IDE
2、安装 Digispark驱动
3、配置 Arduino IDE 对 Digispark开发板的支持
  文件 –> 首选项 –> 附加开发板管理器中添加http://digistump.com/package_digistump_index.json
  工具 –> 开发板 –> 开发板管理器筛选 Digistump AVR Boards 并安装（科学上网爽歪歪）

4、工具 –> 开发板 –> Digispark (Default - 16.5mHz)，选择端口
5、开始CodeText
